### What Microsoft Power Platform examples are available here?

### What Copilot Studio agent examples are available?

Current Copilot Studio projects include an HR onboarding agent that replaces manual email-based onboarding with a Teams AI assistant, and a Microsoft 365 FAQ agent that handles employee self-service support queries. Both are grounded on SharePoint knowledge sources and integrated with Power Automate.

### How does oversharing affect Microsoft 365 Copilot deployments?

Microsoft 365 Copilot respects existing SharePoint permissions — it surfaces content that users already have access to. In most tenants, years of broad sharing (everyone-except-external shares, broken permission inheritance, stale links) means Copilot can surface content that was never intended to be widely visible. The SharePoint Permission Report and Site Scanner projects in this portfolio are designed to help identify and remediate those gaps before Copilot is deployed.

### Are these solutions production ready?

Projects are shared as learning references, solution patterns, and practical starting points. Each includes documentation covering the architecture and build approach. Review your organisation's security requirements, governance policies, and Microsoft licensing before deploying to production.
